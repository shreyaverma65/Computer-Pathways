**Portfolio Website**

**Project Description**

This is a simple **Static Portfolio Website** built using **HTML and CSS**, created as part of the *Computer Science Fundamentals & Career Pathways* assignment.
The website displays personal details, projects, and basic information in a clean and structured layout.

**Technologies Used**

* HTML – Structure of the website
* CSS – Styling and layout
* VS Code– Development environment
* Git & GitHub – Version control and hosting

-->How to Run the Project

1. Download or clone the repository:

   git clone https://github.com/shreyaverma65/Computer-Pathways.git
   
2. Open the project folder in VS Code.
3. Open the file index.html.
4. Right-click → Open with Live Server


-->Features

* Simple and clean user interface
* Organized sections: About, Projects, Footer
* Fully static and easy to customize
* Beginner-friendly code
